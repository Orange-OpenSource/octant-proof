vtr_brs_eq the correspondance between a trace and a set of branches, checking at each level that the branches match the trace. The main idea is to decompose the branches in parts linked to a given atom and then call recursively vtr_brs_eq on the part of the trace corresponding to this atom and on the tail of the branches in the part.
count should be equal to the height of the type tree which is also the maximal length of the branches. When anything wrong is found, the result will be false
